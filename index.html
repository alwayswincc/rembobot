Frequently Asked Questions

Is Rembo really free?

Yes! Rembo is completely free to use for all Discord servers. Invite Rembo now.

How do I set up Rembo?

Simply invite Rembo to your server and use the !help command to get started.

Does Rembo store my data?

Rembo only stores necessary data for functionality and never shares your information.

Can I request new features?

Absolutely! Join our support server and let us know what you'd like to see.
